Профиль администратора

Если в окне авторизации ввести логин и пароль администратора то откроется окно профиля администратора.

В появившемся окне можно увидеть данные авторизовавшегося администратора.

В таблице для отображения сотрудников можно сделать поиск по фамилии, для этого нужно будет ввести символы по которым нужно искать в поле для поиска.

Так же администратор может добавить пособие по нажатию на кнопку "Добавить" и удалить пособие по нажатию на кнопку "Удалить".

По нажатию на кнопку "Профиль" откроется окно для редактирования профиля.
