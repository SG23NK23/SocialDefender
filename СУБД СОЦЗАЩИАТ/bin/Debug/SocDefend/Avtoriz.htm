Авторизация

При запуске программы появляется окно для авторизации пользователя.

В окне есть надпись "Регистрация" при нажатии на которую будет открыто окно для регистрации пользователя:

В открывшемся окне можно зарегестрировать нового пользователя если заполнить поля для вода данных и при нажатии на кнопку "Зарегестрироваться" будет создан аккаунт для нового пользователя.

Если ввести в поля: "Логин" и "Пароль" корректные данные, то можно войти в аккаунт пользователя, администратора или сотрудника .
