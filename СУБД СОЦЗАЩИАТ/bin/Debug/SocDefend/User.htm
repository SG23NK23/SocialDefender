Профиль пользователя

При вводе логина и пароля пользователя под которым нужно авторизоваться откроется окно с профилем пользователя:

В открывшемся окне появятся данные об авторизовавшемся пользователе, где можно просмотреть данные о его пособиях, а так же отредактировать его профиль по нажатию на кнопку "Профиль ".

Так же пользователь можнт оформить пособие, для этого еу нужно выбрать тип пособия и по нажатию на кнопку "Отправить", пособие будет отправлено администратору на одобрение.
