Профиль сотрудника

В появившемся окне профиля сотрудника отображаются данные авторизовавшегося сотрудника, существующие пользователи и заявки пользователей.

По нажатию на кнопку "Сохранить в документ пользователей" будет открыт Excel документ куда будут выведены пользователи.

По нажатию на кнопку "Профиль" будет открыто окно для редактирования профиля.

По нажатию на кнопку "Разрешить" выделенному пользователю в таблце "Заявка", если в поле введена сумма пособия то оно будет одобрено и будет начисляться указанная сумма, а если поле не будет заполенно или заполненно некорректно то пособие не будет одобрено.
