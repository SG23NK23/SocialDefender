Редактирование профиля пользователя

После нажатия на кнопку "Профиль " будет открыто окно для редактирования профиля:

В появившемся окне редактирования профиля можно будет отредактировать данные о пользователе, а чтобы подтвердить редактирование данных нужно будет ввести пароль пользователя и по нажатию на кнопку "Сохранить" если данные корректны и пароль для подтверждения верный то данные успешно изменятся.
